🌻 โครงสร้างผังงาน (Flowchart) ที่เกี่ยวข้อง:

  • ▶ แบบลำดับ (Sequence): ทำงานจากบนลงล่างตามลำดับ (รับข้อมูล -> คิด -> แสดงผล)
  • ▶ แบบมีเงื่อนไข (Condition): ตัดสินใจเลือกทางใดทางหนึ่ง เช่น "รหัสผ่านถูกหรือไม่?"
  • ▶ แบบทำซ้ำ (Loop): วนกลับไปทำงานเดิมจนกว่าจะบรรลุเงื่อนไข เช่น "โหลดข้อมูลจนกว่าจะครบ 100%"

หน่วยรับข้อมูล (Input Unit)

ทำหน้าที่รับข้อมูล คำสั่ง หรือโปรแกรมจากผู้ใช้ส่งเข้าสู่ระบบคอมพิวเตอร์เพื่อนำไปประมวลผล

⌨️ คีย์บอร์ด (Keyboard) - รับตัวอักษร
🖱️ เมาส์ (Mouse) - รับตำแหน่งพิกัด
🎤 ไมโครโฟน (Mic) - รับเสียง
📷 กล้อง (Camera) - รับภาพ/วิดีโอ
🌡️ เซ็นเซอร์ (Sensor) - รับข้อมูลสภาพแวดล้อม (อุณหภูมิ, แสง)

หน่วยประมวลผลกลาง (CPU)

เปรียบเสมือน "สมอง" ของคอมพิวเตอร์ ทำหน้าที่คำนวณ เปรียบเทียบ และควบคุมการทำงานของอุปกรณ์ต่างๆ

องค์ประกอบหลักของ CPU:

  1. ALU (Arithmetic Logic Unit): หน่วยคำนวณและตรรกะ ทำหน้าที่บวก ลบ คูณ หาร และเปรียบเทียบเงื่อนไข (จริง/เท็จ)
  2. CU (Control Unit): หน่วยควบคุม คอยอ่านคำสั่ง สั่งการ และประสานงานอุปกรณ์อื่นๆ
  3. Registers: หน่วยความจำขนาดเล็กมากๆ ใน CPU ที่ทำงานเร็วที่สุด ไว้เก็บข้อมูลชั่วคราวขณะคำนวณ

▶ วัฏจักรการทำงาน (Machine Cycle): Fetch (ดึงคำสั่ง) -> Decode (ถอดรหัส) -> Execute (กระทำการ) -> Store (เก็บผลลัพธ์) **ทำงานเป็น Loop ซ้ำๆ**

หน่วยความจำ (Memory & Storage)

ที่เก็บข้อมูลของระบบ แบ่งออกเป็น 2 ประเภทหลักที่ทำงานร่วมกัน

1. หน่วยความจำหลัก (Primary Memory)

ต้องมีไฟเลี้ยง ข้อมูลจะหายไปเมื่อปิดเครื่อง ทำงานเร็วมาก (ทำงานร่วมกับ CPU โดยตรง)

  • RAM (Random Access Memory): โต๊ะทำงานหลัก เปิดแอปเยอะ RAM ต้องเยอะ
  • ROM (Read Only Memory): อ่านได้อย่างเดียว เก็บโปรแกรมเริ่มต้นเครื่อง (BIOS/UEFI)

2. หน่วยความจำสำรอง (Secondary Storage)

ไม่มีไฟเลี้ยงข้อมูลก็ไม่หาย เป็นคลังเก็บของถาวร ทำงานช้ากว่า RAM

  • HDD (Hard Disk Drive): จานแม่เหล็กหมุน ความจุสูง ราคาถูก ช้า
  • SSD (Solid State Drive): ชิปแฟลช ไม่มีชิ้นส่วนเคลื่อนไหว เร็วมาก ราคาสูงกว่า

หน่วยแสดงผล (Output Unit)

นำผลลัพธ์ที่ CPU ประมวลผลเสร็จแล้ว แปลงกลับมาเป็นรูปแบบที่มนุษย์เข้าใจได้

🖥️

จอภาพ (Monitor)

แสดงผลลัพธ์แบบชั่วคราว (Soft Copy) เป็นภาพหรือข้อความ

🖨️

เครื่องพิมพ์ (Printer)

แสดงผลลัพธ์แบบถาวร (Hard Copy) พิมพ์ลงบนกระดาษ

🔊

ลำโพง (Speaker)

แสดงผลลัพธ์ในรูปแบบของเสียง
